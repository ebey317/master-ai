Setup — one-time, per remote device

  1. Install Tailscale on your remote device.
    • iPhone / iPad — App Store → "Tailscale"
    • Android — Play Store → "Tailscale"
    • Mac / Windows / Linux — tailscale.com/download
  2. Log in with the SAME account you used on Madam-Mary. Both devices must be in the same Tailnet.
  3. Open Tailscale on the remote device. Confirm it shows a green "Connected" / "Active" status.
  4. Open a browser on the remote device and paste the Pupil URL above.
If the page doesn't load: you're probably NOT logged into Tailscale on the remote device. Check the Tailscale app icon — it should be green/blue, not gray. Also verify Madam-Mary is awake (not sleeping) and the stt_server is running (systemctl --user status master-ai-ui.service on the host).
